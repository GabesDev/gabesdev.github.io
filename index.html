Modal Component Documentation

A reusable, accessible modal component following the NB Design System standards.

Structure

  • .modal-container: The fixed-position backdrop.
  • .modal-dialog: The centered dialog box.
  • .modal-header: Contains the title and close button.
  • .modal-body: Main content area.

Accessibility Features

  • role="dialog" and aria-modal="true" labels.
  • Focus trapping within the modal when open.
  • Escape key support for closing.
